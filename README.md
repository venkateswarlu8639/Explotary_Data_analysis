# 📊 Exploratory Data Analysis (EDA) & Dashboard using Excel

## 📌 Project Overview
This project is part of my Data Analytics Internship (Task 2), where I performed Exploratory Data Analysis (EDA) on a sales dataset and created an interactive dashboard using Microsoft Excel.

The objective of this project is to analyze business data, identify patterns, and generate meaningful insights to support decision-making.

---

## 📂 Dataset Description
The dataset contains sales-related information such as:
- Order Date
- Product Name
- Category
- Sales Amount
- Customer/Region details

---

## 🛠 Tools & Techniques Used
- Microsoft Excel
- Pivot Tables
- Data Cleaning
- Data Visualization (Bar Chart, Line Chart, Pie Chart)

---

## 📊 Dashboard Features
The dashboard includes the following key components:

### 🔹 Key Performance Indicators (KPIs)
- Total Sales
- Average Sales
- Total Orders

### 🔹 Visualizations
- Top 10 Products by Sales (Bar Chart)
- Monthly Sales Trend (Line Chart)
- Category-wise Sales Distribution (Pie Chart)

---

## 🔍 Key Insights
- A small number of top products contribute to a large portion of total sales.
- Sales show a noticeable increase during certain months, indicating seasonal trends.
- Categories like Technology and Office Supplies perform strongly in terms of revenue.
- A high number of orders with relatively lower average value suggests frequent customer purchases.
- Sales distribution is uneven across products, indicating opportunities for optimization.
